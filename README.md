# Media Gallery

**Media Gallery lets you create a gallery of your video files directly inside of your notes in obsidian**

> **The plugin for now only supports video galleries**

## Roadmap
- [x] video support
- [ ] Image support
- [ ] Audio support
- [ ] Books support

## Examples
````md
```media-gallery
path: directory/videos
```
````
